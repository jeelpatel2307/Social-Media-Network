# Social Media Network

This project implements a simple social media network in Java. It allows users to create accounts, add friends, remove friends, and view their friend list.

## Features

- User management: Users can create accounts in the social network.
- Friendship management: Users can add and remove friends.
- Friend list: Users can view their list of friends.

## How to Run

1. Make sure you have Java installed on your system.
2. Compile the `SocialNetwork.java` file using the `javac` command: `javac SocialNetwork.java`.
3. Run the compiled file using the `java` command: `java SocialNetwork`.

## Usage

1. When the program starts, you will see a menu with options to create a user account, add a friend, remove a friend, and view friend list.
2. Choose the desired option by entering the corresponding number.
3. Follow the prompts to perform the selected action.
4. You can exit the program by selecting the appropriate option from the menu.

## Example

Here's an example of how to use the social media network:

1. Create user accounts: Alice, Bob, and Charlie.
2. Add friendships between users: Alice adds Bob and Charlie as friends.
3. View friend list: Alice can view her friend list and see that Bob and Charlie are her friends.
4. Remove friendship: Alice removes Bob from her friend list.
5. View updated friend list: Alice can see that Bob is no longer her friend.
